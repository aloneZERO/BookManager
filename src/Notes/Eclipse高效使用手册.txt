常用快捷键：

1. 注释：Ctrl + '/'， 取消：Ctrl + '\'
默认：光标所在行注释
选中部分内容：多行行注释
选中全部内容：区块注释

2. 联想提示：Alt + '/'
默认：自动开启
小细节：
	例如：int a = b. //此时使用快捷键，会联想提示返回值为int类型的方法

3. 快捷输入：系统默认关键字或自定义（这是Java选项下的编辑器中的模板功能）
例如：输入sysout，按下回车会输入System.out.println();

4. 函数定义技巧：
先写好函数主体，然后：/** ，按下回车，会智能生成文档注释
使用系统没有的函数：
 int a = ;
 int b = ;
 int c = add(a,b);
//光标放在add处使用快捷键：Ctrl + 1，提示是否创建此函数，回车确定，自动生成函数主体
//利用tab或shift + tab 可以进行部分修改

5. 快速编辑：
将光标停在目标行，使用快捷键：Ctrl + Alt + C不松，
再按下上下方向键，可以快速复制行内容

上下移动行内容：Alt（不松）+ 方向键（up/down）

6. 快捷键Alt + shift + S，调出菜单快速生出类中Set和Get方法，和其它的自动创建常用方法！

7. 格式化代码格式（其实就是优化排版）
Ctrl + shift + F

8. 运行
Ctrl + Alt + X

9. 调试
Ctrl + Alt + D
F6 执行下一步
F8 执行到下一个断点，如果没有则执行到结束
F5 进入方法内部
Ctrl + Shift + I 查看表达式的结果

10. 双击选中对应代码块
